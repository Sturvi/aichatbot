Порядок выполнения работ и сроки

1
Подготовка (до старта)
Предоставляется первичная база знаний и список из 50–100 типовых вопросов для калибровки качества и тестирования.
2
Старт работ
Вносится аванс 500 манат. Определяется первый подключаемый сервис (канал).
3
Этап A — Core (2–3 недели)
Разрабатывается Core. Предоставляется простой интерфейс для тестирования. При подтверждении качества — переход к следующему этапу.
4
Этап B — Админ‑панель (2–3 недели) + 1‑й сервис (2–4 недели)
Разрабатывается админ‑панель и настраивается первый выбранный сервис. Сдаётся результат: Core + Admin Panel + 1 сервис. Поступает оставшаяся оплата за текущий объём.
5
Этап C — Остальные сервисы
Оставшиеся сервисы подключаются по очереди, каждый в срок 2–4 недели. Запуск следующего сервиса осуществляется после завершения и оплаты предыдущего.
